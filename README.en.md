# rag_nfra

#### Description
该项目是一个基于国家金融监督管理总局（National Financial Regulatory Administration，简称 NFRA）政策法规构建的 RAG， 旨在对 NFRA上的政策法规文件进行智能检索、问答，适用于金融政策、法规检索等场景的智能化应用开发与研究。技术栈：Python 3.12；LangChain 0.3.17；Milvus 2.5.4。
